Requête

  • Req0 -> Afficher les informations sur les voitures de couleur noire
    SELECT *
    FROM `voitures`
    WHERE `couleur` = 'noire'

  • Req1 -> Afficher toutes les informations sur les clients qui habitent à Paris
    SELECT *
    FROM `clients`
    WHERE `ville`='Paris'

  • Req2 -> Afficher les immatriculations et les marques de voitures de couleur noire
    SELECT `immatriculisation` , `marque`
    FROM `voitures`
    WHERE `couleur` = 'noire'

  • Req3 -> Afficher toutes les informations sur les voitures ayant un kilométrages <10000 km
    SELECT *
    FROM `voitures`
    WHERE `kilometrage` <10000

  • Req4 -> Afficher toutes les informations sur les voitures ayant une marque mercedes
    SELECT *
    FROM `voitures`
    WHERE `marque` = 'mercedes'

  • Req5 -> Afficher toutes les dates des locations effectués par le client T200
    SELECT `date_location`
    FROM `locations`
    WHERE `code_client` = 'T200'

  • Req6 -> Afficher les id_location et les date_location concernant la voiture 22 TU 2000
    SELECT `id_location` , `date_location`
    FROM `locations`
    WHERE `immatriculation` = '22 TU 2000'
